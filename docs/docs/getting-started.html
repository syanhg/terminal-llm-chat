Getting Started with Terminal LLM Chat

Introduction

Terminal LLM Chat is a command-line interface for interacting with large language models (LLMs) like GPT-4, Claude, and others. It provides a nostalgic terminal experience combined with modern AI capabilities.

This guide will help you install Terminal LLM Chat and get started with the basic features.

Requirements

  • Python 3.7 or higher
  • pip (Python package manager)
  • API key from at least one of the supported LLM providers:
    • OpenAI
    • Anthropic
    • OpenRouter
    • Local setup with Ollama (no API key required)

Installation

Method 1: Install from PyPI (Recommended)

The easiest way to install Terminal LLM Chat is through pip:

pip install terminal-llm-chat

This will install the package and its dependencies. After installation, you can run the application with:

terminal-chat

Method 2: Install from Source

To install from source:

git clone https://github.com/syanhg/terminal-llm-chat.git
cd terminal-llm-chat
pip install -e .

This installs the package in development mode, allowing you to make changes to the code.

Method 3: Using Docker

If you prefer using Docker:

docker pull syanhg/terminal-llm-chat
docker run -it syanhg/terminal-llm-chat

First Run

When you run Terminal LLM Chat for the first time, it will guide you through a setup process:

  1. Choose an LLM provider

    Select from OpenAI, Anthropic, OpenRouter, or Ollama (local)

  2. Enter your API key

    You'll need to provide an API key for your chosen provider (except for Ollama)

  3. Select a default model

    Choose which model you want to use by default

  4. Choose a theme

    Select from classic green, amber, blue, or custom themes

All settings are saved to a configuration file in your home directory at ~/.config/terminal-llm-chat/config.ini (or equivalent on Windows).

Basic Usage

Starting a Conversation

After launching Terminal LLM Chat, you can start typing your messages directly:

terminal-chat
> Hello, can you help me understand Docker containers?

Using Commands

Terminal LLM Chat supports various commands that begin with a forward slash:

Command Description
/help Display available commands
/clear Clear the screen
/exit Exit the application
/save [filename] Save the current conversation
/load [filename] Load a saved conversation
/system [prompt] View or change the system instructions
/models List available models
/model [name] Switch to a different model
/theme [name] Change the color theme

Advanced Usage

Command-line Arguments

Terminal LLM Chat supports various command-line arguments for advanced use:

terminal-chat --help

Usage: terminal-chat [OPTIONS]

Options:
  --model TEXT          Specify the model to use
  --system TEXT         Set a custom system prompt
  --load TEXT           Load a saved conversation
  --config TEXT         Use a specific config file
  --theme TEXT          Set the color theme
  --api-key TEXT        Set the API key
  --provider TEXT       Set the provider (openai, anthropic, etc.)
  --no-streaming        Disable streaming responses
  --verbose             Enable verbose logging
  --version             Show the version and exit
  --help                Show this message and exit

Multi-line Input

For entering multi-line text or code blocks, you can use triple backticks:

> ```
def hello_world():
    print("Hello, world!")
    
hello_world()
```

This allows you to format your input properly when writing code or structured text.
