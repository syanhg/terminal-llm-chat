Why Terminal LLM Chat?

🚀

Lightning Fast

Optimized for speed and minimal resource usage. No browser overhead, just pure conversation.

🔌

Multiple Providers

Connect to OpenAI, Anthropic, OpenRouter, or run locally with Ollama. Your choice, your control.

🎨

Customizable Themes

Choose from classic green terminal, amber, blue, or create your own color schemes.

💾

Conversation History

Save, load, and manage your chat sessions with simple commands. Never lose an important conversation.

🔒

Privacy Focused

Your API keys and conversations stay on your machine. No tracking, no telemetry.

📋

Code Formatting

Proper syntax highlighting and code formatting in the terminal. Copy code with a single command.

🔧

Extensible

Add custom plugins and commands to extend functionality for your specific needs.

💻

Cross-Platform

Works on Linux, macOS, and Windows. Consistent experience everywhere you code.

Installation

# Install with pip
pip install terminal-llm-chat

# Run the application
terminal-chat
# Pull the Docker image
docker pull syanhg/terminal-llm-chat

# Run the container
docker run -it syanhg/terminal-llm-chat
# Clone the repository
git clone https://github.com/syanhg/terminal-llm-chat.git

# Navigate to the directory
cd terminal-llm-chat

# Install the package
pip install -e .

Requirements

  • Python 3.7+
  • API key from one of:
    • OpenAI
    • Anthropic
    • OpenRouter
  • Or use with Ollama (local)

Usage

Commands

/help Display available commands
/clear Clear the screen
/exit Exit the application
/save [filename] Save the current conversation
/load [filename] Load a saved conversation
/system [prompt] View or change the system instructions
/models List available models
/model [name] Switch to a different model
/theme [name] Change the color theme

See it in action

Seamless AI interaction in your terminal

Terminal LLM Chat provides a distraction-free environment for your AI conversations. Ask questions, generate code, or brainstorm ideas without leaving your terminal.

With streaming responses, you'll see the AI's thinking in real-time as it generates each character, just like you would in a web interface but without the overhead.

Try it out with the installation instructions above, or explore the documentation to learn more about all the features.
